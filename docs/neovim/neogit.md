# Neogit

[neogit](https://github.com/NeogitOrg/neogit) is a Magit-like git interface for Neovim.

## Opening Neogit

| Key | Mode | Description |
|-----|------|-------------|
| `<leader>gn` | n | Open Neogit |

## Status Buffer Keybindings

| Key | Description |
|-----|-------------|
| `?` | Toggle help |
| `q` | Close |
| `<tab>` | Toggle section fold |
| `<enter>` | Go to file / expand |
| `s` | Stage file/hunk |
| `S` | Stage all |
| `u` | Unstage file/hunk |
| `U` | Unstage all |
| `x` | Discard changes |
| `c` | Commit popup |
| `p` | Push popup |
| `P` | Pull popup |
| `r` | Rebase popup |
| `b` | Branch popup |
| `L` | Log popup |
| `d` | Diff popup |
| `Z` | Stash popup |

## Commit Popup

After pressing `c`:

| Key | Description |
|-----|-------------|
| `c` | Create commit |
| `a` | Amend last commit |
| `e` | Extend commit (add staged to last) |
| `f` | Fixup commit |
| `r` | Reword commit message |

## Configuration

Diffview integration is enabled for viewing diffs.
